This is a server for [Where Was That](https://where-was-that-mk-react.web.app/). 

Instructions:
1. Install and or use Node version 20.14.0
2. Install dependencies with command: npm install
3. Initialize server with command: npm start

Source code for Where Was That can be found at:
1. [react web app](https://github.com/michaelkaffel/where-was-that)
2. [react-native mobile app](https://github.com/michaelkaffel/where-was-that-react-native)
